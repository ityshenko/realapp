## ❌ НАЙДЕННЫЕ ПРОБЛЕМЫ

### 1. **НЕСОГЛАСОВАННЫЕ ОТСТУПЫ**

#### Проблема:
- Главная: `py-20` для секций
- Объявления: `py-24` 
- Профиль: `py-8 pt-28`
- Разные `px-4` vs `px-6`

#### Решение:
Унифицировать все отступы:
```css
/* Секции */
section { padding: 5rem 1rem; } /* py-20 px-4 */

/* Контейнеры */
.container { padding-left: 1rem; padding-right: 1rem; }

/* Header padding */
header { padding-top: 10px; padding-bottom: 10px; }
```

---

### 2. **HEADER - РАЗНЫЕ СТИЛИ НА СТРАНИЦАХ**

#### Проблема:
- На главной используется компонент `<Header />`
- На некоторых страницах может быть дублирование

#### Решение:
✅ Проверить что ВСЕ страницы используют `<Header />`

---

### 3. **КАРТОЧКИ ОБЪЯВЛЕНИЙ - НЕТ ЕДИНОГО СТИЛЯ**

#### Проблема:
- ListingCard используется везде ✅
- Но отступы внутри разные: `p-3` vs `p-4` vs `p-6`

#### Решение:
Унифицировать:
```tsx
<div className="p-4"> {/* Единый padding для всех карточек */}
```

---

### 4. **КНОПКИ - РАЗНЫЕ СТИЛИ**

#### Проблема:
- Главная: `px-8 py-3.5`
- Объявления: `px-6 py-3`
- Профиль: `px-5 py-2.5`

#### Решение:
Создать единые классы:
```css
.btn-large { padding: 14px 32px; }
.btn-medium { padding: 12px 24px; }
.btn-small { padding: 10px 20px; }
```

---

### 5. **ТИПОГРАФИКА - СКАЧУЩИЕ РАЗМЕРЫ**

#### Проблема:
- Заголовки: `text-4xl` → `text-3xl` → `text-5xl`
- Нет иерархии

#### Решение:
```css
h1 { font-size: 3rem; }    /* text-5xl */
h2 { font-size: 2.25rem; }  /* text-4xl */
h3 { font-size: 1.875rem; } /* text-3xl */
h4 { font-size: 1.5rem; }   /* text-2xl */
```

---

### 6. **FORM INPUTS - РАЗНЫЕ СТИЛИ**

#### Проблема:
- Логин: `py-3`
- Регистрация: `py-3`
- Создание: `py-4`
- Фильтры: `py-3`

#### Решение:
```css
.input { padding: 12px 16px; } /* py-3 px-4 */
.input-large { padding: 14px 18px; } /* py-3.5 px-4.5 */
```

---

### 7. **GRID LAYOUTS - НЕТ АДАПТИВНОСТИ**

#### Проблема:
- Рекомендованные: `grid-cols-5` только на xl
- На мобильных может быть плохо

#### Решение:
```tsx
className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 xl:grid-cols-5 gap-4"
```

---

### 8. **МОБИЛЬНАЯ ВЕРСИЯ**

#### Проблема:
- Бургер меню есть ✅
- Но некоторые кнопки мелкие (< 44px)

#### Решение:
```css
/* Минимальный размер для тача */
button, a { min-height: 44px; min-width: 44px; }
```
