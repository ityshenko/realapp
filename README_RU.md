# 🏠 РеалТУТ - Недвижимость Донецка

Современный маркетплейс недвижимости в тёмном дизайне (#353535).

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd /Users/ityshenko/Realapp

# Backend
cd backend && npm install

# Web
cd ../web && npm install
```

### 2. Запуск

**Терминал 1 - Backend:**
```bash
cd backend
npm run dev
```
Backend: http://localhost:5000

**Терминал 2 - Web:**
```bash
cd web
npm run dev
```
Web: http://localhost:3000
